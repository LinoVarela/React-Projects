# React Projects

- Repository with some simple react projects learned through the course at [Codedamn](https://codedamn.com/learn/reactjs-projects)

## What was learned (most important topics)

- Structure a React app
- state management
- React hooks
- Creating custoom hooks
- Data Fetching

## Pre-requirements 
 - HTML/CSS
 - Javascript basics

### Create new react app
npx create-react-app "name of the app"

### Start react app

npm start
